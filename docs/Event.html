Event

The event class allows you to create event listeners, that automatically fires when a discord.js event is received

Constructor

new Event(options)

Source:
  • event.js, line 14
Creates a new event handler
Example
const event = new memecord.Event({
  name: 'guildCreate',
  run: function(guild) {
    guild.defaultChannel.send('Thanks for inviting me to your server!');
  }
});
