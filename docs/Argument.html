Argument

The Argument class allows you to create command arguments

Constructor

new Argument(options)

Source:
  • argument.js, line 21
Creates a new argument
Example
const memeNameArgument = new memecord.Argument({
  type: 'string',
  key: 'memeName',
  label: 'Name',
  infinite: true
});

options object The argument options
