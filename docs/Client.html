Client

The Client class allows you to create a new bot instance

Constructor

new Client(options)

Source:
  • client.js, line 24
Creates a new bot instance
Example
const client = new memecord.Client({
  token: 'your-token-here',
  ownerID: 'your-user-id',
  prefix: 'meme!',
  djsOptions: {
    'disableEveryone': true
  }
});

client.run();
Parameters:
Name Type Description
options object Options for the client
Properties
Name Type Attributes Default Description
token string Discord login token for the client
ownerID string | Array.<string> Discord client ID of the owner
prefix string The prefix for the bot to use
selfbot boolean <optional>
 false Whether this bot is a selfbot or not
allowDM boolean <optional>
 false Whether commands via DMs are allowed
allowRecursion boolean <optional>
 false Whether recursion is allowed (REALLY not recommended)
allowBotUsage boolean <optional>
 false Whether other bots may use this bot (not recommended)
logLevel string <optional>
 INFO The log level to use, one of NONE, ERROR, WARNING, INFO, VERBOSE
djsOptions object <optional>
 Options to pass to discord.js, see https://discord.js.org/#/docs/main/stable/typedef/ClientOptions
